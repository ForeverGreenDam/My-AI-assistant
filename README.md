# My-AI-assistant
<p>这是一个前后端分离RAG知识库项目，</p>
主要使用的AI框架是LangChain4j，使用的向量数据库为Redis。

<p>This is a frontend-backend separated RAG knowledge base project, </p>

<p>The project is currently in the early stages of development.</p>
